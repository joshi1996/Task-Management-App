# Task Management App

 A full-stack Task Management Application built with Flutter for the frontend and Node.js for the backend.
The app allows users to sign up, log in, create tasks, view task lists, and assign colors to tasks using a color picker.
Authentication is handled securely using JWT, and the backend is fully containerized with Docker.
